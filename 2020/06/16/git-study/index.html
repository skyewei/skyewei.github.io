使用技巧

  • git status # 查看状态
  • git log # 查看提交日志
  • git reset HEAD file # 取消文件进入暂存区
  • git add -u # 将已经跟踪过的文件，修改后的文都提交到暂存区
  • git checkout – file #取消对文件的更改内容
  • git reset –hard # 取消暂存区文件的所有变更
  • git mv old_name new_name # 快捷对暂存区的文件的 重命名
